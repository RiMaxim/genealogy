https://genealogy.inje.ac.kr - книги с родословными

Инструкция по поиску:

1. Поиск названий и ссылок по фамилии

> python3 ./1.py --surname "현씨" --output-dir "temp_html" --output-file "results.txt"

Скрипт 1.py будет искать на сайте https://genealogy.inje.ac.kr фамилию (например 현) вместе с суффиксом 씨. 씨 — это корейский суффикс вежливости, который добавляется к имени или фамилии человека, чтобы выразить уважение в нейтрально-формальном стиле общения. Будет создана и удалена временная папка с html страницами из которых извлекается информация (название книг и ссылка на скачивание книги). Для сохранения HTML-файлов использовать --keep-html В файле results.txt содежится две колонки: в первой колонке название родословной книги (в названии всегда присуствует фамилия с суффиксом) и во второй колонке ссылка на книгу.
